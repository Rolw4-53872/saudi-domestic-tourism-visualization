# Dataset

## Source
The dataset was obtained from [Kaggle](https://www.kaggle.com/) and contains monthly records of domestic tourism activity in Saudi Arabia from 2018 to 2023.

## How to Download
1. Visit the Kaggle dataset page
2. Download the CSV file
3. Place it in this `data/` directory

## Columns

| Column | Type | Description |
|--------|------|-------------|
| year | int | Year (2018-2023) |
| month | int | Month (1-12) |
| trips | int | Number of trips |
| spendSAR | float | Spending in SAR |
| nights | int | Nights stayed |
| origin_temp | float | Origin province avg temperature |
| destination_temp | float | Destination province avg temperature |
| originProvinceNameEn | str | Origin province name |
| destinationProvinceNameEn | str | Destination province name |
| visitPurposeEn | str | Visit purpose |
